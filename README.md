# Flappy Kiro 🐦

A pixel-art Flappy Bird clone built with pure HTML5/Canvas — no dependencies, no build step.

Play it directly in the browser. No WASM, no Python, no server-side runtime.

## Features

- Pure HTML5 Canvas — zero dependencies
- Pixel-art sprites with tinted background
- Responsive scaling (portrait & landscape)
- Touch, click, and keyboard controls
- Score tracking with digit sprites
- Kiro theme (purple/pink palette)

## Deployment

Pushes to `main` auto-deploy to Railway via Dockerfile.
